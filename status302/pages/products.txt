====Camembert====
[[Camembert]] is a Free, Open source, lightweight IDE, purely written in Fantom.
It's multi-platform and can run on many platforms (Linux, Max, Windows...)

[[camembert | Camembert product page]]

====Axon / Skyspark plugin for Camembert====
This is a [[axon+plugin|Camembert plugin]] which provides many enhanced features for Sparks and Axon projects development.

It provides a much more productive development environment in comparaison to using the web based editor of Skyspark.
